Данный проект представляет собой игру в саппера по сети
Сначала нужно запустить сервер с помощью файла server.exe
Далее можно запускать клиента с помощью файла client.exe
При запуске клиента для подключения к серверу необходимо ввести ip адресс сервера который высвечивается в загаловке окна сервера
Вы подключились. Можете наслаждаться игрой
